# Role — Default

## Purpose
This role is an explicit alias for `Feature-Developer`.

It ensures the agent always operates under a defined role
when no role is explicitly specified by the user.

## Effective role
- Feature-Developer

All priorities, constraints, and mandatory skills are inherited
from `feature_developer.md`.

## Notes
- This role contains no independent behavior.
- Role switching must still follow the rules defined in `AGENTS.md`
  (auto role selection and explicit triggers).
